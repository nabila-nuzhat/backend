- ARP (Address Resolution Protocol)
    - Address Resolution Protocol (ARP) is a fundamental TCP/IP protocol that maps a logical 32-bit IPv4 address to a physical 48-bit MAC (Media Access Control) address.
    - acts as a bridge between the network layer and the data link layer, allowing devices on the same local network to actually route and deliver data.
    Because IP addresses (the destination) change, while MAC addresses (the physical network card) remain permanently fixed, devices rely on ARP to ensure packets reach the right machine.
